# Documentation Index

本目录集中存放仓库内的所有文档/记录，并按主题分类。

## Categories

- `api/session/` - Session API 相关排查、方案与说明
- `build/` - 构建/编译相关记录
- `deployment/` - 部署/刷写/权限下发与验证
- `design/` - 需求、差距分析、架构设计类文档
- `discovery/` - 发现/图管理/协议相关
- `guides/` - 操作指南、快速检查、测试步骤
- `investigation/` - 按日期归档的深度排查
- `ohos/` - OHOS/OpenHarmony 约束、集成、平台相关
- `permission/` - 权限/Token/绕过方案与回归
- `phases/phase2/` - Phase2 阶段文档
- `phases/phase3/` - Phase3 阶段文档
- `pr/` - PR 描述、变更日志、提交流程
- `reports/` - 汇总/结论/最终报告/复盘
- `testing/` - 测试用例、记录、结果
- `misc/` - 未归类但需要保留的文档
- `artifacts/` - 日志与输出留存（体积可能较大）
